Web Browsers

	Linux supports almost all browsers on the market (except some like internet explorer)

	Mozilla is the default browser in linux

	Lynx = text based browser (used for testing websites from inside a text environment)


E mail clients

	programs that allow sending/receiving messages
	before you can use one of those e mail clients, you must supply your email
	server's information like the IP address + credentials

	Mutt = text based e-mail client
